FAQ

คำถามที่พบบ่อย

ต้องมีพื้นฐานเขียนโค้ดไหม?
ไม่จำเป็น ทุกขั้นตอนอธิบายแบบ step-by-step มีคำสั่งให้ copy-paste ตลอด ถ้าเคย copy command จาก YouTube มาแล้ว — ทำได้แน่นอน
ต้องเสียค่าใช้จ่ายอะไรเพิ่มอีกบ้าง?
API key ของ Claude (Anthropic) ประมาณ $5–20/เดือนขึ้นอยู่กับการใช้งาน, VPS ประมาณ $5/เดือน (ถ้าต้องการ 24/7), Stripe ไม่มีค่ารายเดือน (คิดค่าธรรมเนียมต่อ transaction ~2.9%)
OpenClaw ต่างจาก n8n หรือ Zapier ยังไง?
n8n/Zapier คือ workflow automation — คุณออกแบบ flow แล้วมันทำตาม OpenClaw คือ AI ที่คิดเองได้ — มันตัดสินใจ วางแผน และเลือกเครื่องมือให้เหมาะกับสถานการณ์ คู่มือนี้สอนให้ใช้ทั้งสองร่วมกัน
ซื้อแล้วได้รับไฟล์ยังไง?
หลังชำระเงิน จะถูก redirect ไปหน้า thank-you ที่มีปุ่มดาวน์โหลดทันที และยังได้รับอีเมลพร้อม link ถาวรทั้งสองภาษาอีกด้วย
ใช้ได้บน Windows ด้วยไหม?
ได้ — บทที่ 3 ครอบคลุม Windows โดยเฉพาะ รวมถึงวิธีแก้ปัญหา permission และ antivirus ที่พบบ่อย
